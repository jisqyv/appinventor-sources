Wireless Phone Connection Test -- NEWS September 6, 2012

This version changes the way you connect to your phone. The "Connect Wireless" button is gone. Instead under the connect menu there is now a "WiFi" option which does the same thing. Plus, if you exit the Debugger App you can start a new one and connect to the same blocks editor (previously you had to quit and restart the blocks editor). Just start a new debugger app on your phone. The newest version has a "Restart" button. Select "Restart" on your phone and then reselect "WiFi" from the blocks editor connect menu. It should reconnect without you having to enter a new code or scan a barcode.

This version of App Inventor has new features that will be released on the main production server soon. In the meantime you may have problems downloading projects from here and then uploading them to beta.appinventor.mit.edu because your blocks will be too new! This problem should resolve itself in a few days when release v129 is out.
